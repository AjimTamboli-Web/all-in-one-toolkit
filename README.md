## ✨ Features

This project includes **14 fully functional tools**, all built from scratch using React:

| # | Tool | Description |
|---|------|-------------|
| 1 | 📝 **Word Counter** | Count characters, words, sentences, and paragraphs in real-time |
| 2 | 🔐 **Password Generator** | Generate strong, secure passwords with customizable length and complexity |
| 3 | 👶 **Age Calculator** | Calculate exact age in years, months, and days from any birth date |
| 4 | 📲 **QR Code Generator** | Convert any text or URL into a downloadable QR code |
| 5 | 🍕 **Split Bill** | Divide restaurant bills among friends with tip calculation |
| 6 | 🏦 **EMI Calculator** | Calculate monthly loan/mortgage payments with interest breakdown |
| 7 | 🎨 **Color Palette Generator** | Generate random HEX color palettes with one-click copy |
| 8 | ⚖️ **Unit Converter** | Convert between length, weight, temperature, and more |
| 9 | { } **JSON Formatter** | Beautify, minify, and validate JSON data |
| 10 | 🏷️ **Meta Tag Generator** | Generate SEO-ready HTML meta tags for any website |
| 11 | 🧾 **Invoice / GST Calculator** | Calculate inclusive and exclusive GST amounts |
| 12 | 🖼️ **Image Compressor** | Compress images in the browser without uploading to any server |
| 13 | 📄 **PDF Merger** | Merge multiple PDF files into a single document client-side |
| 14 | 💼 **Resume Builder** | Build a professional resume and export it as a PDF |

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/YOUR_USERNAME/all-in-one-toolkit.git

# 2. Navigate to the project folder
cd all-in-one-toolkit

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev

# The app will open at http://localhost:5173
